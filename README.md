# QKA
Q on KonA. You can use `q.k` with the [**misc_fixes**](https://github.com/pahihu/kona/tree/misc_fixes) branch of KonA. To enable APL dyadic word syntax in KonA load with `k -2 q.k`.

Examples:

   / use lambda directly
   1{x+y}2
   3

   / rename + verb
   f: +
   1 f 2
   3

**NOTE**: Use the code! Good luck!
